# arXiv
- 报告号（rn）：一般是科研报告等特有的编号，用于识别报告，如NASA的技术报告有其特定报告号。
- ID列表（id）：是一系列唯一标识符组成的表，像arXiv论文有专属ID，通过id_list参数或id字段限定搜索特定论文。
- 期刊参考（jnr）：指论文发表期刊的相关信息，如期刊名、卷号、期号、页码等，像《Nature》某卷某期某页。
- 物理分类（ph）：在arXiv等平台按物理学领域细分，如hep-th（高能物理-理论）、astro-ph（天体物理学）等。
- ACM分类（ac）：是ACM对计算机领域的分类体系，如计算理论下有自动机理论等分类。
