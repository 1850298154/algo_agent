### 二、各算法包简单说明
| 包名               | 核心算法/用途说明                                                                 |
|--------------------|----------------------------------------------------------------------------------|
| casadi             | 用于数值优化、最优控制的算法库，支持非线性规划、最优控制问题求解，常用于机器人、控制系统等领域 |
| cma                | 实现协方差矩阵自适应进化策略（CMA-ES），一种高效的无约束优化算法，适用于复杂非凸优化问题     |
| cvxpy              | 凸优化问题的建模与求解库，支持线性规划、二次规划、半定规划等凸优化问题，简化优化问题的定义与求解 |
| evotorch           | 基于PyTorch的进化算法库，提供各类进化策略、遗传算法等，适用于机器学习和强化学习中的优化任务   |
| highspy            | HiGHS求解器的Python接口，专注于线性规划（LP）、混合整数线性规划（MILP）等运筹优化问题求解   |
| libsvm             | 经典的支持向量机（SVM）算法实现，用于分类、回归等机器学习任务，是SVM算法的核心实现库         |
| mlrose/mlrose-hiive| 实现模拟退火、遗传算法、随机爬山、神经网络权值优化等启发式算法，专注于组合优化和机器学习优化 |
| networkx           | 图算法库，提供图的创建、遍历、最短路径、最小生成树、社区发现等各类图论相关算法             |
| optuna             | 超参数优化算法库，支持贝叶斯优化、TPE、CMA-ES等算法，用于自动调优机器学习模型的超参数       |
| optunity           | 轻量级的超参数优化库，支持网格搜索、随机搜索、贝叶斯优化等，专注于机器学习模型的参数调优     |
| ortools            | Google开源的运筹优化库，提供整数规划、线性规划、约束规划、路径规划（如TSP）、调度算法等     |
| pyomo              | 运筹优化建模语言，支持线性规划、非线性规划、整数规划等，可对接各类求解器（如Gurobi、CPLEX）|
| scikit-learn       | 机器学习算法集成库，包含分类、回归、聚类、降维、模型选择等经典算法（如决策树、随机森林、KNN等） |
| scikit-opt         | 启发式优化算法库，实现遗传算法、粒子群优化、模拟退火、蚁群算法等，适用于各类组合/数值优化问题 |
| scipy              | 科学计算核心库，包含数值积分、优化、插值、线性代数、统计等基础算法，是算法开发的基础工具     |
| statsmodels        | 统计建模与分析库，提供线性回归、时间序列分析（ARIMA、VAR等）、假设检验等统计算法           |

### 总结
1. 这些算法包覆盖**优化算法**（凸优化、进化算法、启发式优化）、**运筹算法**（线性/整数规划、调度）、**机器学习算法**（SVM、经典分类回归）、**图算法**、**统计算法**五大类核心算法场景。
2. 核心用途集中在：超参数调优（optuna/optunity）、运筹优化（ortools/pyomo/highspy）、启发式优化（mlrose/scikit-opt/cma）、机器学习（scikit-learn/libsvm）、数值优化（casadi/cvxpy/scipy）。
